# Expense Summary

A simple JavaScript expense summary utility that calculates:
- total expense amount
- totals by category
- the largest expense item

## Project URL
https://roadmap.sh/projects/js-expense-summary

## Files
- `index.js` — expense data and summary functions

## Features
- Calculate the total of all expenses
- Calculate total spending for a specific category
- Find the largest expense entry

## Run
```bash
node "javascript/Expense Summary/index.js"
```
